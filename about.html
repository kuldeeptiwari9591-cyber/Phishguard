About PhishGuard

A final-year project focused on phishing detection, explainability, and user awareness.

Developer

Kuldeep Tiwari
B.Sc. Information Technology – Final Year

This project was developed to demonstrate how rule-based analysis and threat intelligence can help users identify suspicious websites without relying on black-box machine learning.

System Architecture

  • Frontend: HTML, CSS, JavaScript for user interaction and explanations
  • Backend: Python Flask for URL analysis and API handling
  • Logic Engine: Rule-based phishing detection using domain, SSL, and reputation signals
  • Database: MongoDB for scan history storage

Detection Approach

PhishGuard uses an explainable, rule-based approach instead of machine learning. Each verdict is derived from visible signals such as:

  • Domain age and ownership
  • SSL certificate validity
  • Domain reputation and threat intelligence
  • Brand impersonation indicators

Ethical Disclaimer

This system does not guarantee website safety and should not be used as the sole decision-making tool. It is designed for educational and awareness purposes only.
